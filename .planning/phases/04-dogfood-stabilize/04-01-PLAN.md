## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Information Disclosure | `/api/latency` endpoint | accept | Endpoint returns only aggregate statistics (p95, avg, max, count); no sensitive event data exposed. Dashboard is local-only. |
| T-04-02 | Denial of Service | Latency tracker ring buffer | accept | Tracker maintains max 2048 samples and discards oldest; no unbounded memory growth. Heavy load simply updates recent history. |
